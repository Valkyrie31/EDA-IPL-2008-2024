#  IPL Exploratory Data Analysis (2008–2024)

##  Overview
This project explores IPL match data from 2008 to 2024 using Python, Pandas, Matplotlib, and Seaborn. The goal is to uncover trends in team performance, player achievements, match outcomes, and overall tournament evolution through data-driven analysis and visualizations.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

##  Key Insights
- Identified the most successful IPL franchises based on total wins and win percentage.
- Analyzed the impact of toss decisions on match outcomes.
- Explored season-wise trends in scoring and match competitiveness.
- Discovered the highest run scorers and leading wicket takers in IPL history.
- Examined venue-wise performance patterns and winning trends.
- Analyzed Player of the Match awards to identify the most influential players.
- Compared team performances across different IPL seasons.
